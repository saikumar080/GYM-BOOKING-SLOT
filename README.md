# 🏋️ Gym Slot Booking App

A full-stack gym slot booking system built with React, Node.js, and MySQL.

## 🚀 Live Demo
👉 [Click here to open the app](https://gym-booking-slots.vercel.app)

## ⚙️ Backend API
👉 https://gym-booking-slot.onrender.com/api/slots

## 📦 Tech Stack
- Frontend: React, CSS
- Backend: Node.js, Express
- Database: MySQL (Railway)
- Deployment: Vercel + Render

## 🔧 Features
- View available gym slots
- Book a slot (max 10 per slot)
- Cancel a booking

## 🛠️ Run Locally
### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm start
